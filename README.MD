# Implementação de Produto Tensorial


Neste repositório estão contidos dois arquivos .c

- main.c: Código principal com a função int main(), responsável pela leitura dos arquivos .txt e a leitura das matrizes escritas
- biblioteca.c: Descrição do código completo da função contida na biblioteca.h

# Instruções para execução

É importante que tenha o Makefile instalado para a compilação automática do código

```bash
sudo apt install build-essential
```

Para compilar:

```bash
make clean
make
```

Para executar:

```bash
./tensor arquivo1.txt arquivo2.txt... arquivoN.txt
```

Este repositório foi desenvolvido em uma máquina virtual Linux-Mint
